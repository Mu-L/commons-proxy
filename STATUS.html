1. INTRODUCTION

The Proxy package contains a set of Java classes that facilitate the creation of dynamic proxies.

2. DEPENDENCIES

The Collections package is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.3 or later)
  • Apache Commons Logging (Version 1.0.4 or later) - for logging, optional for deployment
  • AOP Alliance (Version 1.0 or later) - for method interceptor support, optional for deployment
  • JUnit Testing Framework (Version 3.8.1 or later) - for unit tests only, not required for deployment
  • Apache Axis (Version 1.2.1 or later) - used for testing JaxRpcProvider, not required for deployment (JaxRpcProvider requires some JAX-RPC implementation at runtime if you wish to use it)
  • Apache Commons Discovery (Version 0.2 or later) - used for testing JaxRpcProvider (required by Axis), not required for deployment
  • Javassist (Version 3.0 or later) - used by JavassistProxyFactory for dynamic proxy generation, not required for deployment unless you wish to use JavassistProxyFactory
  • CGLIB (Version 2.0.2 or later) - used by CglibProxyFactory for dynamic proxy generation, not required for deployment unless you wish to use CglibProxyFactory
  • Burlap (Version 2.1.7 or later) - used by BurlapProvider, not required for deployment unless you wish to use BurlapProvider
  • Hessian (Version 3.0.1 or later) - used by HessianProvider, not required for deployment unless you wish to use HessianProvider
  • Apache XML-RPC (Version 2.0 or later) - used to test XmlRpcInvoker, not required for deployment unless you wish to use XmlRpcInvoker
  • Doug Lea's Concurrency API (Version 1.3.4 or later) - used by ExecutorInterceptor, not required for deployment unless you wish to use ExecutorInterceptor
